This is a exercise based on abstraction and basic object oriented programming in javaScript.

In this exercise we made a stopwatch object which works like a stopwatch. To know more refer to the code and play with in the console.

You can use followwing command in the console to play with it:

    Note:- you have to create an object in the console before using these methods or functions.

  1. start():-This will start the stopwatch
  2. stop():- This will stop the stopwatch.
  3. reset():- This will reset the stopwatch.
  4. duration:- This will show you the duration till the stopwatch run.
